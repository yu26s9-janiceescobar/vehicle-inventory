# Vehicle Inventory

## Description
A Java console application that allows users to browse and manage a vehicle inventory.

## How to Run
1. Clone the repository
2. Open in IntelliJ
3. Run Main.java

## Features
- View all vehicles in inventory
- Search by make/model of vehicle 
- Search by color of vehicle
- Search by price range
- Add vehicle to inventory

## Technologies Used
- Java
- IntelliJ IDEA
